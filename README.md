It provides dedicated dashboards for **Admin**, **Trainer**, **Student**, **Accountant**, and **Manager**, along with AI-powered modules that help analyze student performance, improve placement readiness, and support institutional decision-making.

# рџљЂ Getting Started

## Clone Repository

```bash
git clone https://github.com/Sarthak794/AI-Powered-ERP-System.git
```

---

## Open Project

Import the project into:

- Spring Tool Suite (STS)
- IntelliJ IDEA
- Eclipse

---

## Configure Database

Create a MySQL database:

```sql
CREATE DATABASE edu_erp;
```

Copy:

```text
application-example.properties
```

Rename it to:

```text
application.properties
```

Update:

- Database Username
- Database Password
- Razorpay Keys
- Email Configuration

---

## Run Application

```bash
mvn spring-boot:run
```

Open:

```
http://localhost:9095
```
