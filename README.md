# 📌 Overview

The **AI-Powered Education ERP System** is a comprehensive web-based solution designed to automate and simplify educational institution management.

The system provides dedicated dashboards and features for different user roles while integrating AI-powered analytics to improve decision-making and student performance tracking.

# 🚀 Installation

```bash
git clone https://github.com/Sarthak794/AI-Powered-ERP-System.git
```

Import into STS or IntelliJ.

Configure

```
application.properties
```

Run

```
mvn spring-boot:run
```
